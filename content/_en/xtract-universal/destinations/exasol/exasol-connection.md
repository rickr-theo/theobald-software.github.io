**Connection string**<br>
Enter the name or IP of the DB2 server and the port number. 

**User Id / Password**<br>
Enter the user ID and password of the database user.

**Schema**<br>
Enter the schema of the database.
  
**Test Connection**<br>
Checks the database connection.
